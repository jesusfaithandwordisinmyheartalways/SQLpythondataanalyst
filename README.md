
# SQL & Python Data Analyst Project

This project demonstrates a complete beginner-level data analysis workflow using **Python, Pandas, SQL, and Tableau Public**. 
The purpose is to clean, explore, and visualize sales data for a small business scenario.




### Tableau Dashboard — Sales Analysis
Interactive dashboard on Tableau Public:

https://public.tableau.com/app/profile/andrew.johnson8782/viz/SalesAnalysisAndrewJohnson/Dashboard1








## Project Structure

- **data/**  
  - `raw_sales_data.csv` → Original messy sales data.  
  - `cleaned_sales_data.csv` → Cleaned and processed CSV generated by Python.

- **python/**  
  - `data_cleaning.py` → Python script that performs:
    1. Load raw sales CSV.
    2. Fill missing values (Customer names, Quantity).
    3. Drop duplicate rows.
    4. Recalculate total revenue per row.
    5. Save cleaned CSV.

- **sql/**  
  - `create_tables.sql` → SQL script to create the `sales` table.  
  - `insert_data.sql` → SQL script to insert sample sales data into the table.  
  - `queries.sql` → Example queries to calculate total revenue, sales by product, and customer order counts.

- **dashboards/**  
  - `tableau_workbook.twbx` → Tableau Public workbook displaying:
    - KPI cards (Total Revenue, Total Orders, Top Product)
    - Sales trends over time
    - Product performance
    - Customer insights
    - Interactive filters
